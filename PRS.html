Introduction

This module is designed and developed in order to generate products and services recommendations on banking user data. The module predicts the most probable product or service purchase by the user . This information can be leveraged and used to push products or services to the user for better personlized UI experience.

Architecture




This module is built on ML algorithms to find the behaviour similarity of the customer with other customer data and provide a prediction of recommendations that user might be willing to purchase in the future.
We have used collaborative filtering algorithm in order to formulate the similarities between the customers. The recommendation results can be structured to promote for future.



Softwares Used

1. Python
2. Streamlit
3. REST API
